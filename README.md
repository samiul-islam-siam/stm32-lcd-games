# 🎮 STM32 LCD Games

A collection of classic games implemented on a **16×2 HD44780 LCD** using the **STM32 Nucleo-F446RE** development board.

Built with **STM32CubeMX**, **STM32CubeIDE**, and the **STM32 HAL Driver**, these projects demonstrate how custom characters, animations, and creative rendering techniques can transform a simple character LCD into a gaming platform.
